<p>
  📉 Matplotlib es una biblioteca de Python para la creación de gráficos y visualizaciones de datos. 
  🎨 Es extremadamente flexible y permite a los usuarios generar una amplia variedad de gráficos, 
  desde simples líneas y puntos hasta complejas visualizaciones tridimensionales. Matplotlib es una 
  herramienta fundamental en el análisis y la presentación de datos científicos.
</p>
